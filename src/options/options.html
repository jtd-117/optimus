About:

Optimus is a Chrome extension designed to maximise browser productivity by offering the following tools:

  • Timer
  • Website Blocking

When the timer is activated via the play button︎︎, user-selected websites on the block list cannot be accessed until the timer is reset via the reset button.


  • Pressing the pause button does not grant back access to blocked websites. This can only be done via the reset button.
  • Websites cannot be removed from the block list while the timer is active

Timer Settings:

Timer Settings can be accessed via the `Timer` button under the `Menu` navigation section within the options page, or under the extension popup.


Here, users can adjust the number of hours, minutes and seconds the timer will run for during a session.

Website Blocking:

The Block List can be accessed via the `Block` button under the `Menu` navigation section within the options page, or under the extension popup.


Here, users can add and delete websites from the block list. Also note that you can add blocked websites via the popup page.


Websites on this list cannot be accessed when the timer is active.


For details on how to block specific pages or entire websites, visit Block settings page.

Settings Transfer:

The Transfer page allows users to import or export their configured settings via a single json file.


If users have several devices with Google Chrome installed as a browser, settings can be restored via the aforementioned JSON file.

GitHub Repository:

The codebase for this project can be found under the optimus repository on GitHub.


All relevant files including code, documentation, issues, etc... can be found in the aforementioned repository.
