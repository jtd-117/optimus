About

Optimus is a Chrome extension designed to maximise browser productivity by offering the following tools:

  • Session Timer
  • Website Blocking

When the timer is activated via the popup play button︎︎, user-selected websites on the block list cannot be accessed until the timer has elapsed, or reset via the reset button.

Session Timer

The session timer serves as a countdown timer for how long websites on the block list will be inaccessible for.


The session timer is accessed via the extension popup and has the following buttons:

  • Play - start the session countdown
  • Pause - freeze the session but continue to keep the websites blocked
  • Reset - to restart the session timer

Session Timer Settings can be accessed via the `Timer` button under the `Menu` navigation section within the options page, or under the extension popup.


Here, users can adjust the number of hours, minutes and seconds the timer will run for during a session.

Website Blocking

Websites on the block list cannot be accessed via the Chrome broswer or edited within website blocking settings while the session timer is active.


The block list can be found via the `Blocking` button under the `Menu` navigation section within the options page, or under the extension popup.


Here, users can add or delete websites from the block list. Also note that you can add blocked websites via the popup page.


When completing the website blocking form, there is a dropdown that shows how to enter website URLs properly.

Settings Transfer

The Transfer Settings page allows users to import or export their configured settings via a single json file.


If users have several devices with Google Chrome installed as a browser, along with the Optimus extension installed, settings can be restored via an exported JSON file.

Extension Permissions

With user data and privacy being a sensitive topic, this project is committed to being transparent in the types of user data that this Chrome extension can accessed.


This Chrome extension requires the following permissions to function:

  • Permission 1
  • Permission 2

Rules For Entering URLs
  • one line per website
  • Do not enter www.
  • Blocking an entire website: website
  • Blocking a website page: website/page
  • Blocking all subpages: website/*
