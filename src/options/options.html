About

Optimus is a Chrome extension designed to maximise browser productivity by offering the following tools:

  • Session Timer
  • Website Blocking

When the timer is activated via the popup play button︎︎, user-selected websites on the block list cannot be accessed until the timer has elapsed, or reset.

Session Timer

The session timer serves as a countdown timer for how long websites on the block list will be inaccessible for.


The session timer is accessed via the extension popup and has the following buttons:

  • Play - start the session countdown
  • Pause - freeze the session but continue to keep the websites blocked
  • Reset - to restart the session timer

Session timer settings can be accessed via the 'Blocking' button on the options page.


Here, users can adjust the number of hours, minutes and seconds the timer will run for.


Note that session timer settings cannot be edited while the timer is active.

Website Blocking

Websites on the block list cannot be accessed via the Chrome broswer or edited while the session timer is active.


The block list can be found via the 'Blocking' button in the the options page.


Here, users can add or delete websites from the block list.


The extension popup also allows users to block websites they are currently on.


When completing the website blocking form, please adhere to website blocking guide when entering links.

Settings Transfer

The 'Transfer Settings' page allows users to import or export their configured settings via a single json file.


If users have several devices with Google Chrome installed as a browser, along with the Optimus extension installed, settings can be restored via an exported JSON file.


Users can also reset timer and website blocking settings to it's original state.

Website Blocking Settings

Edit:

Click the 'Edit Block List' button and complete the form to update blocked websites.

Edit Blocked Website(s):

Click the 'Submit' button to update the block list.

Rules For Entering URLs
  • one line per website
  • Do not enter www.
  • Do not enter https:// or http://
  • Blocking an entire website: websiteName
  • Blocking a website page: websiteName/page
  • Blocking all subpages: websiteName/*
